Setting it up

You will need to have the following installed:

  • Python (2 or 3) and PIP
  • Either virtualenv or pyvenv

Creating your project

Create a virtual environment and activate it. Then run:


$ pip install wagtail
$ wagtail start my_website

Running it


$ cd my_website
$ python manage.py migrate              # Setup the database
$ python manage.py createsuperuser      # Create yourself an admin account
$ python manage.py runserver            # Run the development server

The site should now be running on: http://localhost:8000
